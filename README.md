## 🧠 ¿Qué es Git?

Git es un sistema de control de versiones.

Permite:

-   Guardar el historial completo de cambios
-   Volver atrás si algo se rompe
-   Trabajar varias personas a la vez
-   Probar cosas sin miedo

En lugar de tener: proyecto_final_v2_definitivo_bueno_ahora_si.zip

Con Git tienes historial limpio + ramas + control real.

## 🛠 Requisitos Previos

### En Mac

brew install git

### En Windows

Descargar desde: https://git-scm.com/

Verifica que funciona: git --version

------------------------------------------------------------------------

## 📦 1. Clonar el repositorio

git clone https://github.com/chals23/web-rookies.git cd web-rookies

------------------------------------------------------------------------

## 🌿 2. Crear tu primera rama

git checkout -b mi-primera-rama

------------------------------------------------------------------------

## ✏️ 3. Hacer tu primer cambio

1.  Abre un archivo del proyecto
2.  Modifica algo simple
3.  Guarda el archivo

Comprueba cambios: git status

------------------------------------------------------------------------

## 💾 4. Hacer tu primer commit

git add . git commit -m "Cambio texto de bienvenida"

------------------------------------------------------------------------

## ☁️ 5. Subir los cambios a GitHub

git push origin mi-primera-rama

------------------------------------------------------------------------

## 🔁 Flujo básico de trabajo

1.  git pull
2.  Crear rama
3.  Modificar código
4.  git add
5.  git commit
6.  git push

------------------------------------------------------------------------

## ⚠️ Errores comunes

### Nothing to commit

No has cambiado nada o no hiciste git add.

### Merge conflict

Dos personas han cambiado la misma línea.

### Detached HEAD

Estás en un commit específico, no en una rama.
